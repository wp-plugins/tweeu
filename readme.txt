== Description ==

TweeU is a Wordpress plugin based on Tweetly Updater that updates Twitter when you create or edit a blog entry, it uses the buh.bz service for short urls.
It can use your tags and categories to add hashtags to your tweet.
TweeU uses buh.bz, instead of bit.ly. There is no need for authentication on buh.bz.

== Installation ==

TweeU requires PHP 5 >= 5.2.0 and the curl module for PHP.

- Unzip the release archive to your Wordpress plugin directory. cd /path/to/blog/wp-content/plugins; unzip tweeu.zip
- Activate the plugin on the plugin management page.
- Go to Settings -> TweeU and fill in your twitter login information.
