== Description ==

TweeU is a Wordpress plugin that updates Twitter when you create or edit a blog entry, it uses the buh.bz service for short urls based on Tweetly Updater.
It can use your tags and categories to add hashtags to your tweet.

== Installation ==

TweeU requires PHP 5 >= 5.2.0 and the curl module for PHP.

* Unzip the release archive to yout Wordpress plugin directory
* Activate the plugin on the plugin management page
* Go to _Settings->TweeU_ and fill in your twitter login information. 
* Enjoy...
